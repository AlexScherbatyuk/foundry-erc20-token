## ERC20 Tokem

**Simple ERC20 token based on Openzeppelin contracts with 100% test coverage.**

## Usage
```shell
$ make install
```

### Build

```shell
$ make build
```

### Test

```shell
$ make test
```

### Format

```shell
$ make fmt
```

### Gas Snapshots

```shell
$ make snapshot
```

### Anvil

```shell
$ anvil
```

### Deploy

```shell
$ forge deploy
```


```shell
$ forge deploy-sepolia
```
